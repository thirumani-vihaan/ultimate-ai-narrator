# Ultimate AI Narrator

Peblo Flutter Developer Intern Challenge — **The AI Story Buddy & Quiz Component**.

A joyful, kid-first mobile experience: an AI Story Buddy narrates a short story aloud,
then reveals a fully **data-driven** interactive quiz. Built with performance on
mid-range (~3GB RAM) Android devices as a hard constraint.
